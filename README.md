# Mena's Art World (v1.2)

A front-end creative digital gallery web concept designed to host custom illustrations, animation showreels, and merchandise links for a digital artist portfolio.

## Tech Stack & Highlights
Semantic Structure: Leverages HTML5 layouts to cleanly handle diverse multimedia types.
Embedded Media Frameworks:** Supports inline video rendering and raw audio streams (`.mp3`/`.ogg`) for custom animations and soundtracks.
Internal Anchor Navigation: Fully functional page-jump architecture (`#Home`, `#Media`, `#Contacts`) within a unified page design.
